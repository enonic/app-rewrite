  • This is the list of rewrite-rules for a selected vhost
  • The import-format supports a simplified version of Apache mod_rewrite and CSV
  • The Pattern is a regexp-pattern defining which incoming URLs should be affected by the rule
  • The Substitution is where should the matching requests be sent. A substitution could be:
    • A modified version of the pattern (regexp-substitution)
    • A full URL redirect statement
  • Examples:
    • Redirect one url to another
      /test/pattern/url -> /redirected/anotherUrl
    • Redirect children to another base-path, e.g /test/pattern/product to /redirected/product
      /test/pattern/(.*) -> /redirected/$1
    • Redirect to external
      /test/pattern/.* -> https://enonic.com
